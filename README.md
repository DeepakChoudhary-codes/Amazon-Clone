# Amazon Clone

## Description
A responsive Amazon-inspired e-commerce website built using HTML, CSS, and JavaScript. The project demonstrates frontend development concepts including dynamic product rendering, category filtering, shopping cart functionality, and user authentication using Local Storage.

## Features
- User Login & Signup
- Product Search
- Category Filtering
- Shopping Cart
- Product Details Page
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage

## How to Run
1. Download or clone this repository.
2. Open `index.html` in your browser.
